What artistic style should the image gen prompt target?

E-ink 1-bit dithering destroys photorealistic images. High-contrast illustration styles survive best. Which direction?

A

Woodcut / Engraving

Explicit prompt guidance: "woodcut print", "etching", "linocut". Bold black lines, crosshatching, no gradients. Looks intentionally period-correct on e-ink. Best for fantasy themes.

B

Ink Illustration / Pen-and-Ink

"Pen and ink illustration", "ink wash", "black and white manga". High contrast, defined outlines. More flexible — characters, landscapes, architecture all work.

C

Silhouette / Graphic

Solid black silhouettes against white (or vice versa). Stark, dramatic. No detail to lose in dithering. Skyline, landscape, creature profiles.

D

Let the spec define a testing phase

Don't commit to one style — build in an experiment step where we generate 3-4 samples with different style prompts, convert to BMP, and pick what looks best on the actual hardware.
